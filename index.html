  • Fish

    2d game only css

    Codepen link

    Задумка анимированной 2d игры средствами jade, css3 и svg.

    Первый уровень. Нужно кликать по рыбкам, чтобы набрать баллы. Каждая пойманная рыбка приносит определенное количество баллов.

    Основной принцип работы:

    Инициализация счетчика:
    body { counter-reset: game; } * counter-reset применяется для родительского элемента

    Для хранения значения используется свойство counter-increment: counterName counterIncrement; с именем счетчика и значением. Наиболее подходящими для хранения данного значения являются флажки - input:checkbox с их глобальным атрибутом :checked.

    #fish-1:checked { counter-increment: counterName 5; }

    Для отображения значения счетчика можно использовать псевдоэлемент :after/:before

    #fish-1:checked:after { content: "+5"}
